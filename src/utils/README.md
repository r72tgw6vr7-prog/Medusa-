# Utils Directory

Utility functions and helpers used throughout the Medusa application.

## Utilities

- **analytics.ts** - Google Analytics and tracking utilities
- **error-reporting.ts** - Error logging and reporting
- **gallery-manifest-loader.ts** - Gallery data loading
- **gallery-utils.ts** - Gallery helper functions
- **glassmorphism.ts** - Glass effect utilities
- **image-utils.ts** - Image processing and optimization
- **image-validator.ts** - Image validation utilities
- **apply-fixes.sh** - Shell script for applying fixes

## Notes

- Utils are pure functions without side effects
- Reusable across components and pages
- Follow TypeScript best practices
