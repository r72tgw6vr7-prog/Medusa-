# Hooks Directory

Custom React hooks for the Medusa application.

## Hooks

- **useAnalytics.ts** - Analytics tracking hook
- **useKeyboardNav.ts** - Keyboard navigation utilities
- **useMediaQuery.ts** - Responsive design utilities
- **useReducedMotion.ts** - Accessibility motion preferences
- **useScrollProgress.ts** - Scroll progress tracking (0-1)
- **useSectionTransition.ts** - Section enter/exit transitions with Intersection Observer

## Notes

- All hooks follow React hooks rules
- Include TypeScript types
- Handle edge cases and cleanup
