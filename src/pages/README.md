# Pages Directory

This directory contains all page components for the Medusa website.

## Pages

- **HomePage.tsx** - Main landing page
- **ArtistsPage.tsx** - Artists showcase page
- **GalleryPage.tsx** - Photo gallery page
- **ServicesTestPage.tsx** - Services page
- **ContactPage.tsx** - Contact form and information
- **FAQPageNew.tsx** - Frequently asked questions
- **Legal Pages**:
  - **AGBPage.tsx** - Terms and Conditions
  - **DatenschutzPage.tsx** - Privacy Policy
  - **ImpressumPage.tsx** - Legal Notice
  - **LegalPage.tsx** - Additional legal information
- **AftercarePage.tsx** - Aftercare instructions
- **NotFoundPage.tsx** - 404 error page

## Notes

- All pages use the Medusa Design System
- Pages are routed via the main router configuration
- Each page should maintain consistent styling and UX patterns
