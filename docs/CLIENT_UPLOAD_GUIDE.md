# Anleitung zum Hochladen von Fotos

## Einloggen

1. Öffnen Sie die Admin-Seite unter `/admin`
2. Geben Sie folgende Anmeldedaten ein:
   - Email: salon@example.de
   - Passwort: password123

## Fotos hochladen

1. Wählen Sie die passende Kategorie aus dem Dropdown-Menü:
   - Tattoos
   - Piercings
   - Studio

2. Fotos hochladen - zwei Möglichkeiten:
   - Ziehen Sie die Fotos direkt in das markierte Feld
   - Klicken Sie auf das Feld und wählen Sie die Fotos aus

3. Warten Sie, bis der Upload abgeschlossen ist
   - Ein Ladeindikator zeigt den Fortschritt an
   - Bei erfolgreichen Upload verschwindet der Ladeindikator

## Wichtige Hinweise

- Unterstützte Dateiformate: JPG, PNG, WEBP
- Die Fotos erscheinen automatisch in der Galerie
- Bei Fehlern erscheint eine rote Fehlermeldung
- Melden Sie sich nach dem Hochladen ab
