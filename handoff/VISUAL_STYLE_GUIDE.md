### Color Usage Rules

#### ✅ DO

- Use **Deep Black (#000000)** for all backgrounds
- Use **Base White (#FFFFFF)** for primary text
- Use **Brand Primary (#4e2a3f)** for CTAs and important actions
- Use **Brand Accent (#c9a9bd)** for borders and subtle highlights
- Use opacity variants for depth (e.g., `rgba(78, 42, 63, 0.3)`)

#### ❌ DON'T

- Never use off-brand colors (blue, green, red, yellow)
- Don't use gray backgrounds (only pure black)
- Don't use off-white or cream (only pure white)
- Don't mix gold and magenta themes

### Typography Best Practices

#### ✅ DO

- Use Playfair Display for emotional impact (headlines)
- Use Inter for clarity and readability (body text)
- Maintain consistent line heights (1.5x font size minimum)
- Use proper hierarchy (H1 > H2 > H3 > Body)
- Ensure text contrast ≥ 4.5:1 (WCAG AA)

#### ❌ DON'T

- Don't use more than 2 font families
- Don't use font sizes smaller than 14px
- Don't use all caps for long text (max 3-4 words)
- Don't use tight line heights (< 1.4x)
- Don't use low-contrast text colors

## 📸 Photography Guidelines

### Image Style

#### ✅ DO

- Use high-quality, professional photography
- Maintain consistent lighting and color grading
- Focus on details and craftsmanship
- Use dark or neutral backgrounds
- Show tattoos in context (on skin)
- Include variety of styles and subjects

#### ❌ DON'T

- Use low-resolution or blurry images
- Include watermarks or text overlays
- Use bright, distracting backgrounds
- Show incomplete or healing work
- Use inconsistent color grading
- Include client faces without consent

### Image Specifications

```
Portfolio Images:
  - Format: WebP (with JPEG fallback)
  - Dimensions: 1200x1600px (portrait) or 1600x1200px (landscape)
  - Quality: 85-90%
  - File Size: < 200KB (WebP)
  - Aspect Ratio: 3:4 or 4:3

Artist Portraits:
  - Format: WebP (with JPEG fallback)
  - Dimensions: 800x800px (square)
  - Quality: 90%
  - File Size: < 150KB
  - Aspect Ratio: 1:1

Studio Photos:
  - Format: WebP (with JPEG fallback)
  - Dimensions: 1920x1080px
  - Quality: 90%
  - File Size: < 300KB
  - Aspect Ratio: 16:9
```

## 🚫 Common Violations

### ❌ What NOT to Do

```
1. Using Off-Brand Colors
   ❌ background: #3498db (blue)
   ✅ background: #4e2a3f (brand primary)

2. Wrong Font Families
   ❌ font-family: Arial, Helvetica
   ✅ font-family: 'Inter', sans-serif

3. Arbitrary Spacing
   ❌ margin: 15px
   ✅ margin: 16px (space-4, on 8px grid)

4. Drop Shadows Instead of Glows
   ❌ box-shadow: 0 4px 8px rgba(0, 0, 0, 0.3)
   ✅ box-shadow: 0 0 20px rgba(125, 49, 93, 0.3)

5. Small Touch Targets
   ❌ button { height: 32px }
   ✅ button { min-height: 48px }

6. Poor Contrast
   ❌ color: #888 on #000 (2.6:1 ratio)
   ✅ color: #FFF on #000 (21:1 ratio)
```

## 📱 Responsive Design

### Breakpoint Strategy

```
Mobile First Approach:

Base (320px - 639px):
  - Single column layouts
  - Stacked navigation
  - Full-width cards
  - Touch-optimized (48px targets)

Tablet (640px - 1023px):
  - 2-column grids
  - Horizontal navigation
  - Larger typography
  - Increased spacing

Desktop (1024px - 1439px):
  - 3-4 column grids
  - Full navigation
  - Optimal typography
  - Standard spacing

Wide (1440px+):
  - Max-width container (1440px)
  - Centered content
  - Generous spacing
  - Large imagery
```
